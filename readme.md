The main function is _fizzbuzz_. This is used and called in the test and also in the _debug_ function. It has two 
parameters the user input and the arguments. Those are introduced if running from the commnad line and console or for 
test from the file as direct parameters.

The general workflow of the function is based on storing the result in an array of string, that makes it easier later 
to processes (e.g. to reverse or insert in certain position). The result is a string, that is processed by an auxiliary 
function _arrayToString_. 

The logic of the program follows the instruction provided for different numbers, with a clean visual for each divisor 
that also needs to be on the flags of the program.
